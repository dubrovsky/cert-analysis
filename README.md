# cert-analysis
Web Repository of X509 certificates - CER, CRT, CRL, P7B

Backend:
- Java 8
- Spring Boot
- Spring Security
- Spring Data JPA + Bean Validation
- Hibernate
- Jackson
- REST
- JSON 
- Oracle
- X.509 certificates
- Ehcache 
- Thymeleaf for email templates
- JavaMail
- Tomcat
- Maven

Frontend:
- Angular
- Angular CLI
- TypeScript
- RxJS 
- PrimeNG
- npm
